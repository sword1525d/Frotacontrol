# **App Name**: Frotacontrol

## Core Features:

- Login Screen: User authentication and login screen.

## Style Guidelines:

- Primary color: Strong blue (#3498db), inspired by concepts of efficiency, trustworthiness, and logistics.
- Background color: Very light blue (#eaf2ff).
- Accent color: Slightly darker shade of blue (#2980b9).
- Font pairing: 'Poppins' for headlines (sans-serif), and 'PT Sans' for body (sans-serif).
- Modern and minimalist icons to represent vehicle status and functions.
- Clean and intuitive mobile-first layout with easy navigation.
- Smooth transitions and loading animations to enhance user experience.
